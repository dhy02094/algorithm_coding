유클리드 호제법(gcd(최대공약수))

```python
def gcd(x,y):

	if y ==0 :
		return x
	else :
		return gcd(y, x%y)
```

```python
from collections import Counter

stat = [1,2,3,3,2,3,3,3,4,5]

print(cnt)
```
